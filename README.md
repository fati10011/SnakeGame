# SnakeGame
The classic snake game made in Java with Swing.
The player moves a snake using keyboard strokes and tries to survive and get the highest score possible.

# What I learnt:
<ul>
  <li>Start a project from scratch</li>
  <li>Java Swing</li>
</ul>
